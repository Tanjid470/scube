# SCM Flutter Dashboard App

A **modern, responsive Flutter application** for visualizing and managing data efficiently. The app provides interactive dashboards, progress indicators, and detailed views to help users track revenue, daily data, and other key metrics.

> **Note:** This project was developed as an **Interview Task Project** to demonstrate Flutter UI/UX skills, modular architecture, and responsive design.

---

## Overview

This project is designed to demonstrate **clean UI/UX, modular Flutter architecture, and responsive design principles**. It features a highly interactive interface with tabs, custom widgets, and dynamic elements for better user engagement.

---

## Key Features

- **Interactive Dashboards**
  - Semi-circle progress indicators for visual representation of metrics.
  - Revenue and daily data summaries with detailed cards.
  - Customizable display units and values.

- **Tabbed Navigation**
  - Organized sections for Revenue, Data, and Custom Date views.
  - Smooth navigation with sub-tabs for Today’s Data and other metrics.

- **Custom Widgets**
  - Reusable UI components like cards, buttons, input fields, and progress indicators.
  - Multi-layered circle indicators and colored status indicators.
  - Expandable sections with dynamic arrow animations.

- **Responsive Design**
  - Scales seamlessly across different devices and screen sizes.
  - Adaptive layouts and elements for tablets and mobile screens.

- **Enhanced User Interaction**
  - Expand/collapse sections with smooth animations.
  - Dynamic icons that reflect state changes.
  - User-friendly form validation and input feedback.
